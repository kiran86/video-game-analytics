
# Video Game Sales and Engagement Analysis

## Overview
This project analyzes video game **sales**, **user engagement**, and **ratings** data to uncover trends in game popularity, platform performance, and genre success.  
The goal is to combine **sales data** with **engagement metrics** to generate actionable insights for game developers, publishers, and marketers.
